고급 대시 보드 검토

{{ page.description }}

고급 대시보드는 계정에서 실적이 가장 좋고 최악의 동영상에 대한 개요를 제공합니다. 계정에 게시된 동영상이 10개 미만인 경우 실적이 가장 우수한 동영상 10개만 표시됩니다. 이 대시 보드는비디오 벤치 마크 점수 (VBS)각 비디오에 대해. VBS는 동영상의 성능을 동일한 유형의 다른 유사한 동영상의 성능과 비교합니다. 고급 대시 보드는 비디오에 대한 비디오 콘텐츠 유형 메타 데이터 필드가 활성화 된 경우에만 표시됩니다.

대시보드 페이지를 보려면계기반탐색 헤더에서.

시간 범위 선택

대시보드의 기간을 지정하려면 빠른 날짜를 클릭하거나 캘린더 컨트롤을 사용할 수 있습니다.

빠른 날짜 사용

일련의 빠른 날짜가 페이지 상단에 있습니다. 사용할 수 있는 빠른 날짜는 다음과 같습니다.

  • 오늘
  • 어제
  • 지난 7일
  • 지난 30일
  • 이번 달 - 달력 월 누계 - 예를 들어, 오늘이 3월 17일 경우 이 보고서에는 3월 1일부터 3월 17일이 표시됩니다.
  • 지난 달 - 이전 달력 월 - 예를 들어, 오늘이 2월 14일 경우 이 보고서에는 1월 1일부터 1월 31일이 표시됩니다.

빠른 날짜를 클릭하여 대시보드를 업데이트합니다.

캘린더 컨트롤 사용

캘린더 컨트롤을 클릭하여 시작 날짜와 종료 날짜를 선택할 수도 있습니다.

적용을클릭하여 대시보드를 업데이트합니다.

보고 시간대

대시보드의 데이터는 계정의 보고 시간대를 사용하여 표시됩니다. 시간대가 캘린더 컨트롤 아래에 표시됩니다.

보고 시간대는 보고 데이터에 대한 일 경계를 계산하는 데 사용됩니다. 보고 시간대 설정을 변경하면 향후 데이터에만 영향을 미치며 변경 사항은 소급 적용되지 않습니다. 이 설정을 변경하면 변경 사항이 적용된 일 경계를 살펴볼 때 데이터에 평평한 지점이나 스파이크가 나타날 수 있습니다. 이 설정에 대한 업데이트는 즉시 적용되지 않을 수 있으며 시스템에서 변경 사항을 처리할 때까지 분석 데이터는 이전 설정을 계속 참조할 수 있습니다.

보고 시간대는계정 설정의 일부로 구성할 수있습니다.

대시보드에서 사용되는 용어

다음 비디오 용어를 검토하면 고급 대시보드를 보기 전에 유용합니다.

  • 비디오 시작 - 비디오가 재생되기 시작한 횟수입니다.
  • 동영상 노출 수 - 비디오가 로드되어 상호 작용할 준비가 된 횟수
  • 재생 속도 - 실제로 재생된 동영상의 비율 (동영상 조회수/동영상 노출수)
  • 비디오 벤치 마크 점수 (VBS) -VBS는 동영상의 성능을 동일한 유형의 다른 유사한 동영상의 성능과 비교합니다 ( VBS는 어떻게 계산됩니까? )
  • 평균 조회률 - 평균 조회 시간/비디오 길이

대시보드 분석 검토

페이지 본문에는 고급 분석이 표시됩니다. 페이지는최고 성능 비디오과최악의 성능 비디오섹션. 각 섹션은 동일한 정보를 표시합니다.

참고: 동영상이 10개 미만인 계정의 경우최고의 공연 비디오섹션이 표시됩니다.
참고: 지정된 동영상만동영상 콘텐츠 유형대시보드 보기에 포함됩니다.

그만큼최고 성능 비디오목록에는 VBS가 가장 높은 5 개의 비디오가 표시됩니다. 목록에는 다음이 포함됩니다.

  • 비디오 이름
  • 비디오 콘텐츠 유형
  • 게시 날짜
  • 비디오 벤치 마크 점수

목록에서 비디오를 선택하면 비디오에 대한 VBS 세부 정보가 표시됩니다.

클릭이 비디오에 대한 추가 분석열 것이다해석학선택한 비디오에 대한 자세한 분석이 포함된 페이지.

재생률 그래프에는 선택한 비디오의 재생률과 동일한 비디오 콘텐츠 유형을 가진 다른 비디오의 재생률이 표시됩니다. 동일한 유형의 다른 비디오와 비교 한 비디오의 재생률을 기반으로 성능 등급이 표시됩니다.

성능 평가 VBS 범위
큰 80 ~ 100
좋은 60 ~ 79
평균 40 ~ 59
낮은 20-39
가난한 0 ~ 19
