비디오 속성 편집

{{ page.description }}
 

비디오의 속성을 보고 편집하려면 다음과 같이 하십시오.

  1. 딸깍 하는 소리비디오탐색 헤더에서.
  2. 비디오 제목을 클릭하여 비디오 속성 페이지를 엽니다.

비디오 속성 페이지 상단에 있는 버튼은 다음과 같은 기능을 제공합니다.

  • 새 포스터 및 썸네일 이미지 캡처
  • 비디오 게시
  • 비디오 비활성화/활성화
  • 비디오 삭제

비디오가 활성화되면비활성화버튼을 사용하여 비디오를 비활성화할 수 있습니다. 비디오가 비활성화된 경우, 활성화버튼을 사용하여 비디오를 활성화할 수 있습니다. 비활성화된 비디오는 재생할 수 없습니다.

페이지 상단의 미리보기 플레이어를 사용하여 비디오를 볼 수 있습니다.

비디오 속성은 다음 섹션으로 구성됩니다.

  • 비디오 정보
  • 이미지
  • 비디오 소스 파일
  • 텍스트 트랙

비디오 속성을 편집하려면 섹션에서편집버튼을 클릭하거나 섹션을 두 번 클릭합니다.

비디오 정보

비디오 정보 섹션은 기본 비디오 속성을 보고 편집하는 데 사용할 수 있습니다.

등록 정보를 편집하려면편집버튼을 클릭하거나 섹션을 두 번 클릭합니다. 다음과 같은 속성을 사용할 수 있습니다.

  • 이름 - 255자를 넘지 않습니다.
  • ID - 시스템에 할당된 비디오 ID (편집할 수 없음)
  • 간단한 설명 - 250자를 넘지 않습니다.
  • 비디오 콘텐츠 유형 -비디오 유형; 다음 중 하나 일 수 있습니다. 더 알아보기... ) :
    • 브랜드/회사 프로모션
    • 제품 개요
    • 제품 세부 정보 및 사용
    • 후기
    • 이벤트 프로모션
    • 기업 문화
    • 웨비나
    • 인터뷰/프레젠테이션
  • 태그 - 하나 이상의 태그를 쉼표로 구분합니다. 태그는 비디오 검색을 용이하게합니다.
    • 태그는 최대 128자까지 입력할 수 있습니다. 태그가 128자를 초과하는 경우 변경 사항을 저장하려고 할 때 오류가 표시됩니다.
    • 태그에는 쉼표를 포함할 수 없습니다. 예를 들어 “지붕, 다락방+앞마당”태그는 지원되지 않습니다.
    • 동영상에는 총 1200개의 태그를 가질 수 없습니다.
    • 대소문자, 대문자) 만 다른 태그는 동일한 태그로 처리되며 대소문자가 원래 태그의 태그와 자동으로 일치합니다.
    • 태그를 입력한 후반점키 또는시작하다 . 추가된 태그는 컬러 박스에 표시됩니다.
  • 나누는 -동영상이이 계정으로 공유 된 경우 공유 정보가 표시됩니다.

저장을클릭하여 변경 사항을 저장합니다.

이미지

이미지 섹션에는 비디오의 포스터 및 축소판 이미지가 표시됩니다. 포스터와 썸네일 이미지는 중간 지점에서 소스 비디오에서 스냅샷을 생성하여 자동으로 만들어집니다. 캡처된 기본 이미지가 마음에 들지 않으면 다음을 수행할 수 있습니다. 새로운 이미지 캡처또는다른 이미지 업로드포스터 및 썸네일로 사용합니다.

비디오 소스 파일

비디오 소스 파일 섹션은 비디오 소스 파일을 대체하는 데 사용할 수 있습니다. 이 기능은 소스 파일이 업로드된 후 편집된 경우에 유용합니다. 소스 파일을 교체하면 캡처된 기존 비디오 메타데이터 및 분석이 모두 유지됩니다.

소스 파일을 바꾸려면 [ 바꾸기 ] 를 클릭하고 새 소스 파일을 찾아서 선택합니다.

텍스트 트랙

텍스트 트랙 섹션은 비디오에 텍스트 트랙을 추가하는 데 사용할 수 있습니다. 텍스트 트랙 구성에 대한 자세한 내용은 비디오에 텍스트 트랙추가를 참조하십시오 .
