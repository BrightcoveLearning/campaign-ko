이메일을 통해 비디오 조회 유도

{{ page.description }}

Brightcove Campaign의 게시 기능™랜딩 페이지와 이메일을 사용하여 비디오를 배포하는 데 필요한 내장 코드와 이미지를 쉽게 생성하는 데 사용할 수 있습니다. 일반적으로 비디오 소스 코드는 랜딩 페이지에 붙여 넣은 다음 랜딩 페이지에 대한 링크가 이메일을 통해 전송됩니다. 이 주제에 설명 된 프로세스는 마케팅 자동화 플랫폼 (MAP)이 다음과 같은 경우에 가장 적합합니다. 아니사용되고 있습니다. MAP를 사용하는 경우 Brightcove는 일반게시 프로세스 .

랜딩 페이지 생성

랜딩 페이지는 제품 또는 서비스를 홍보하기위한 마케팅 캠페인의 일부로 사용할 수있는 독립형 웹 페이지입니다. 랜딩 페이지에 비디오를 추가하는 것은 시청자 참여를 높이고 시청자를 구매자로 전환하는 좋은 방법입니다. 랜딩 페이지를 생성하는 기능을 제공하는 여러 공급 업체가 있습니다. 랜딩 페이지는 마케팅 자동화 플랫폼 소프트웨어를 사용하거나 개발자 리소스가있는 경우 사용자 지정 HTML을 사용하여 수동으로 만들 수도 있습니다. 랜딩 페이지가 생성되면 Brightcove Campaign을 사용하여 랜딩 페이지에 비디오를 임베드 할 수 있습니다.

소스 코드 생성

동영상이 랜딩 페이지에서 재생 되려면 소스 코드가 필요합니다. 비디오의 임베드 코드를 복사하려면 다음 단계를 수행하십시오.

  1. 비디오페이지를 엽니다.
  2. 사용하려는 비디오 옆의 확인란을 클릭하여 비디오를 선택합니다.
  3. 게시를 클릭합니다 .
  4. 복사코드 삽입동영상을 표시 할 랜딩 페이지에 붙여 넣습니다.
  5. 랜딩 페이지의 변경 사항을 저장하십시오.
  6. 랜딩 페이지를 게시하고 랜딩 페이지를 볼 수있는 URL이 있는지 확인하십시오.

포스터 이미지 복사

대부분의 플랫폼에서 비디오를 이메일에 직접’삽입하는 것은 불가능합니다. 이는 브라이트코브의 한계가 아닙니다. 거의 모든 최신 이메일 시스템은 보안상의 이유로 비디오 플레이어 임베드 코드를 제거합니다. 하지만 티저 텍스트와 포스터 이미지가 포함된 이메일을 만들어 콘텐츠에 대한 인지도 및 트래픽을 유도할 수 있습니다. 포스터 이미지는 비디오 플레이어처럼 보이고 방문 페이지로 리디렉션되는 클릭 가능한 링크입니다.

비디오의 포스터 이미지를 복사하려면 다음 단계를 수행하십시오.

  1. 페이지 상단의이메일탭을 클릭합니다.
  2. 복사를 클릭하여 포스터 이미지를 복사합니다.
  3. 이메일을 보낼 도구에 이미지를 붙여넣습니다.
  4. 이미지에서 랜딩 페이지로 연결되는 하이퍼링크를 만듭니다. 이를 수행하는 프로세스는 사용중인 이메일 프로그램에 따라 다릅니다.

이메일이 발송되면 독자는 이메일의 일부로 비디오를 보게되고 이미지를 클릭하면 비디오를 볼 수있는 랜딩 페이지로 이동합니다.
