비디오 관리

{{ page.description }}

계정의 비디오에 액세스하려면브라이트코브 캠페인™클릭비디오탐색 헤더에서.

페이지 본문에는 다음과 같은 내용이 표시됩니다.

  • 비디오 썸네일 및 지속 시간
  • 비디오 이름
  • 비디오 ID - 시스템에 할당된 비디오 ID (모든 계정에서 고유함)
  • 상태
    • - 비디오가 활성 상태임을 나타냅니다.
    • - 동영상이 비활성 상태임을 나타냅니다 (재생할 수 없음)
  • 비디오 콘텐츠 유형
  • 생성된 날짜 및 시간 및 사용자 ID
  • 업데이트된 날짜 및 시간 및 사용자 ID

동영상 썸네일 옆에 있는 체크박스를 클릭한 다음 Shift 키를 누른 채 다른 체크박스를 클릭하면 여러 동영상을 선택할 수 있습니다. 열 머리글을 클릭하여 비디오 목록을 정렬할 수 있습니다. 동영상 페이지는 “고정” 정렬을 사용하며 정렬 선택을 기억합니다. 탐색하고 다시 돌아오거나 다시 로그인하면 비디오 페이지가 마지막으로 선택한 정렬로 기본 설정됩니다.

페이지 상단의 버튼 막대는 다음을 수행할 수 있는 기능을 제공합니다.

  • 업로드 -계정에 동영상 업로드
  • 게시 -웹 페이지 또는 이메일에 동영상 게시
  • 콘텐츠 유형 편집 -동영상에 동영상 콘텐츠 유형 할당
  • 활성화 / 비활성화 -동영상 활성화 또는 비활성화
  • 지우다 -계정에서 동영상 제거

비디오 속성을 편집하려면 비디오 이름을 클릭하여 비디오 속성 페이지를 엽니다. 보다비디오 속성 편집비디오 속성 편집에 대한 정보는

동영상 게시

Brightcove Campaign의 게시 기능을 사용하면 마케팅 자동화 플랫폼 (MAP)을 사용하여 비디오를 배포하는 데 필요한 포함 코드와 이미지를 쉽게 생성 할 수 있습니다. 비디오 게시에 대한 자세한 내용은비디오 게시 .

콘텐츠 유형 편집

그만큼비디오 콘텐츠 유형비디오 페이지의 열에는 비디오에 대한 비디오 콘텐츠 유형이 표시됩니다. Brightcove는 비디오 콘텐츠 유형을 사용하여 유사한 비디오의 분석을 컴파일하고 Brightcove의 비디오 벤치 마크 점수 (VBS) 및 맞춤형 조언을 포함하는보다 강력한 분석을 제공합니다. 비디오 콘텐츠 유형을 하나 이상의 비디오에 할당하려면 :

  1. 동영상 옆에있는 확인란을 클릭하여 하나 이상의 동영상을 선택합니다.
  2. 딸깍 하는 소리콘텐츠 유형 편집 .
  3. 선택한 동영상의 콘텐츠 유형을 선택하세요.
  4. 저장을 클릭합니다 .
참고: 브라이트코브는 모든 비디오에 비디오 콘텐츠 유형을 할당할 것을 권장하지만계정 설정의 일부로 이 필드를 비활성화할 수있습니다.

동영상 업로드

계정에 비디오를 업로드하는 방법에 대한 자세한 내용은비디오 업로드 .

비디오 활성화 및 비활성화

비디오 페이지의상태열은 비디오가활성화또는비활성화되었는지 여부를 나타냅니다 .

  • - 비디오가 활성 상태임을 나타냅니다 (비디오를 재생할 수 있음).
  • - 동영상이 비활성 상태임을 나타냅니다 (비디오를 재생할 수 없음)

에활성화비디오를 선택하고활성화 . 에비활성화비디오를 선택하고비활성화 . 상태열이 표시되지 않으면 열 선택기를 사용합니다. 를 클릭하여 선택합니다.

다른 계정에 비디오 공유

생성 및 업데이트된 날짜 및 시간 정보

동영상을 생성 및/또는 업데이트한 사용자의 사용자 ID가 [생성 날짜/업데이트 날짜] 열에 표시됩니다. 작업을 수행한 사용자가 계정에서 제거된 경우에도 해당 ID는 계속 표시됩니다.

동영상을 변경해도 사용자 ID와 사용자 ID가 표시되지 않는 경우가 기록되지 않는 경우가 있습니다.

  • 비디오 소스가 교체되면 비디오의 업데이트된 날짜가 업데이트되지만 작업을 시작한 사용자는 기록되지 않습니다.
  • 이미지 (포스터/썸네일) 및/또는 텍스트 트랙이 업데이트되면 비디오의 업데이트 날짜가 업데이트되지만 작업을 시작한 사용자는 기록되지 않습니다.
