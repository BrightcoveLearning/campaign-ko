클라이언트 측 API를 사용하여 HubSpot에 연결

{{ page.description }}

브라이트코브 캠페인은 HubSpot과 통합되도록 구성할 수 있습니다. 통합이 구성되면 브라이트코브 캠페인을 사용하여 게시된 비디오의 비디오 참여 데이터가 HubSpot에 동기화됩니다. HubSpot에 들어가면 이 데이터는 세분화, 캠페인 로직, 보고서, 잠재 고객 점수 및 커뮤니케이션을 개인화하는 데 사용할 수 있습니다.

참고: REST API를 사용하여 HubSpot과 함께 사용하도록 Brightcove Campaign을 구성하는 방법에 대한 정보는 다음을 참조하십시오. REST API를 사용하여 Marketo에 연결 .

Video Cloud-HubSpot 통합의 주요 기능 중 일부는 다음과 같습니다.

  • 데이터 형식 - 데이터 보기는 사용자 지정타임라인 이벤트 유형으로 제공됩니다 . HubSpot에 복제되는 데이터는 다음과 같습니다.
    • 비디오 이름
    • 비디오 ID
    • 계정 ID (비디오 클라우드)
    • 페이지 URL (참조 페이지의 URL)
    • 플레이어 ID (비디오 클라우드 플레이어)
    • 시청률 (25 %, 50 %, 75 %, 95 %)
  • 데이터 전송 - 데이터 보기는 자동화된 프로세스를 사용하여 HubSpot으로 전송됩니다. 그러나 HubSpot 플랫폼의 일부 보고서는 새로 고치는 데 약간의 시간이 걸릴 수 있습니다.
  • 데이터 보존 -수집 된 데이터는 선택적으로 최대 6 개월 동안 Brightcove Campaign에 보관할 수 있습니다.

연결 설정

HubSpot에 대한 연결을 설정하려면 다음 단계를 따르십시오.

참고: 연결 프로세스에서 브라우저에서 팝업 창이 열립니다. 연결을 설정하기 전에 브라우저 팝업 차단기를 비활성화해야 합니다.

브라이트코브 캠페인과 허브스팟 간의 연결 확립

  1. 브라이트코브 캠페인에로그인합니다.
  2. 탐색 헤더에서 톱니바퀴 아이콘 () 를 누른 다음 MAP 연결을선택합니다. 현재 연결 목록 (있는 경우) 이 나타납니다.
  3. 새 연결추가를 클릭합니다 .
  4. 드롭다운 목록에서 HubSpot을 선택합니다.
  5. 선택대용량 (클라이언트 측 API)통합 유형.
  6. HubSpot 입력계정 ID .
  7. 허브스팟에연결 을 클릭합니다 .
  8. HubSpot에 로그인하라는 메시지가 표시됩니다. 이메일주소와비밀번호를 입력합니다 . 로그인을 클릭합니다 .
  9. 로그인이 성공하면브라이트코브 비디오커넥트가 계정에 액세스할 수 있도록 허용하라는 메시지가 표시될 수 있습니다. 액세스권한 부여를 클릭합니다 .

HubSpot에 대한 연결이 설정되고 사용할 준비가 되었습니다. 기본적으로연결 상태는활성화됩니다 . 연결 상태는 비디오 참여 데이터가 Marketo 로 전송되는지여부를 결정합니다.

  • 활성화 - 데이터가 수집되어 HubSpot으로 전송됩니다.
  • 사용 안 함 - 데이터가 수집되지 않음 (통합이 비활성화됨)
