페이지 내용

    오라클 엘로쿠아에 접속하기

    이 항목에서는 Oracle Eloqua와 함께 사용할 브라이트코브 캠페인™ 을 구성하는 방법에 대해 알아봅니다.

    브라이트코브 캠페인은 Oracle Eloqua와 통합되도록 구성할 수 있습니다. 통합이 구성되면 브라이트코브 캠페인을 사용하여 게시된 비디오에 대한 비디오 참여 데이터가 Eloqua와 동기화됩니다. Eloqua에 들어가면 이 데이터는 세분화, 캠페인 로직, 보고서, 잠재 고객 점수 및 커뮤니케이션 개인화에 사용될 수 있습니다.

    요구 사항

    Oracle Eloqua와 함께 브라이트코브 캠페인을 사용하려면 다음을 수행해야 합니다.

    • 로그인 할 Oracle Eloqua 10 및 자격 증명이 있어야합니다. https://login.eloqua.com
      참고: 이러한 자격 증명은 Eloqua 로그인 페이지에 입력되며 브라이트코브에 직접 제공되지 않습니다.
    • Eloqua에 로그인하는 데 사용되는 자격 증명을고급 사용자-마케팅그룹이고 다음 권한이 있습니다.
      • API 사용
      • 클라우드 응용 프로그램 설치
      • 마케팅 캠페인 설정

    특징

    브라이트코브 캠페인 — Oracle Eloqua 통합의 주요 기능은 다음과 같습니다.

    • 데이터 형식 -비디오 시청 데이터는 Oracle Eloqua의 사용자 정의 데이터 개체 (CDO)로 전달됩니다. Eloqua에 복제되는 데이터는 다음과 같습니다.
      • 비디오 이름
      • 비디오 ID
      • 브라이트코브 계정 ID
      • 참조 페이지의 페이지 URL
      • Brightcove 플레이어 ID
      • 감시률 (%)
    • 데이터 전송 -15 분 이상 지난 데이터보기는 Brightcove Campaign에서 시간당 4 번 Eloqua로 전송됩니다. 즉, 데이터를 보는 데 Eloqua에 표시되는 데 최대 30분이 걸릴 수 있습니다.

    연결 설정

    Eloqua와의 통합을 구성하기 전에 새 브라우저 창을 열고 Eloqua 계정에서 수동으로 로그아웃하는 것이 좋습니다. 이렇게 하면 통합을 구성할 때 Eloqua 계정에 로그인하라는 메시지가 표시되고 올바른 계정을 사용하고 있는지 확인할 수 있습니다.

    참고: 연결 프로세스를 통해 브라우저에서 팝업 창이 열립니다. 연결을 설정하기 전에 브라우저 팝업 차단기를 비활성화해야합니다.

    브라이트코브 캠페인과 Oracle Eloqua 간의 연결을 설정하려면 다음 단계를 따르십시오.

    1. 브라이트코브 캠페인에 로그인합니다.
    2. 탐색 헤더에서 톱니바퀴 아이콘 ( ) 을 클릭한 다음MAP 연결을클릭합니다. 현재 연결 목록 (있는 경우) 이 나타납니다.
    3. 새 연결 추가를클릭합니다.
    4. 고르다 Eloqua드롭 다운 목록에서.
    5. Eloqua에 연결 을클릭합니다.
    6. Eloqua에 로그인하라는 메시지가 표시됩니다. Eloqua를 입력하십시오회사 이름 , 사용자 이름과암호 . 로그인을클릭합니다.
    7. 로그인이 성공하면Eloqua용 브라이트코브 비디오 커넥트앱을 설치하라는 메시지가 표시됩니다. 수락 및 설치를클릭합니다.
    8. 계정에 대해 Eloqua를 활성화하라는 메시지가 표시됩니다. 활성화를 클릭합니다.
    9. 통합 사용에 대한 약관을 검토 한 다음나는 동의한다 .
    10. Eloqua용 브라이트코브 비디오 커넥트를 통해 Eloqua계정에 액세스할 수 있도록 허용하라는 메시지가 표시됩니다. 로그인을클릭합니다.
    11. 회사 이름과 사용자 이름을 확인하고 [ 수락 ] 을 클릭합니다.
    12. 설치를 계속하라는 메시지가 표시됩니다. 딸깍 하는 소리설정 계속 .
    13. 연결 세부 정보 표시를 확인합니다.

    이제 Eloqua에 대한 연결이 설정되고 사용할 준비가 되었습니다. 기본적으로연결 상태는사용 가능으로 설정됩니다 . 그만큼연결 상태비디오 참여 데이터가 Eloqua로 전송되는지 확인합니다.

    • 사용 - 데이터가 수집되어 Eloqua로 전송됩니다.
    • 사용 안 함 - 데이터가 수집되지 않습니다 (통합이 비활성화됨).

    저장을 클릭하여 변경 사항을 저장합니다. 통합을 제거하려면삭제를누릅니다.
