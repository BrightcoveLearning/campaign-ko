Salesforce에 연결

{{ page.description }}

Brightcove Campaign은 Salesforce와 통합되도록 구성 할 수 있습니다. 통합이 구성되면 Brightcove Campaign을 사용하여 게시 된 비디오에 대한 비디오 참여 데이터가 Salesforce에 동기화됩니다. Salesforce에서이 데이터는 세분화, 캠페인 논리, 보고서, 리드 스코어링 및 개인화 커뮤니케이션에 사용될 수 있습니다.

요구 사항

Salesforce에서 Brightcove Campaign 모듈을 사용하려면 다음을 수행해야합니다.

  • Salesforce 계정이 있으면 Salesforce 샌드 박스 계정이 지원되지 않습니다.
  • Salesforce.com 계정이있는 API 사용보기 데이터를 보내도록 선택한 Salesforce 개체를 읽고 쓸 수있는 역할에 속합니다.
  • Pardot 통합을 사용하려면 다음에 설명 된 단계를 완료해야합니다. Pardot 통합을위한 사용자 지정 Salesforce 개체 만들기이야기.

특징

Brightcove Campaign-Salesforce 통합의 주요 기능 중 일부는 다음과 같습니다.

  • 데이터 형식 -보기 데이터는 Salesforce 개체에 매핑됩니다.
  • 데이터 전송 -보기 데이터는 Brightcove Campaign에서 Salesforce로 매시간 전송됩니다.
  • 리드 양식 -시청자가 동영상을 볼 때 자동으로 표시되는 Salesforce 양식을 만듭니다. 자세한 내용은 Salesforce에서 리드 양식 만들기 .

연결 설정

다음 단계에 따라 Brightcove Campaign과 Salesforce 간의 연결을 설정하십시오.

참고: 연결 프로세스에서 브라우저에서 팝업 창이 열립니다. 연결을 설정하기 전에 브라우저 팝업 차단기를 비활성화해야 합니다.
참고: Brightcove는 Brightcove Campaign과의 통합을 구성할 때 사용할 전용 Salesforce 사용자 계정을 생성할 것을 권장합니다.
  1. 브라이트코브 캠페인에로그인합니다.
  2. 탐색 헤더에서 톱니바퀴 아이콘 () 를 누른 다음 MAP 연결을선택합니다. 현재 연결 목록 (있는 경우) 이 나타납니다.
  3. 새 연결추가를 클릭합니다 .
  4. 고르다영업드롭 다운 목록에서 Salesforce에 연결 .
  5. 로그인 화면이 나타나면 Salesforce 사용자 이름및암호를 입력합니다 . 로그인을 클릭합니다 .
  6. 로그인이 성공하면 Salesforce용브라이트코브 비디오 연결앱을 설치하라는 메시지가 표시됩니다. 허용을 클릭합니다 .
  7. 성공적으로 연결되었다는 메시지가 수신되는지 확인하십시오. Brightcove Campaign으로 다시 리디렉션됩니다.
  8. (선택 사항) 연결 이름을 입력합니다 .
  9. 통합 유형을 선택하십시오.
    • Pardot -Pardot은 시청자를 식별하는 데 사용됩니다. 아래의 Pardot 통합구성섹션을 참조하십시오.
    • 설명서 -당신의 개발 직원은 로직을 추가 할 책임이 있습니다. 시청자 식별 .
  10. 매핑 유형을선택하고 데이터 매핑을구성합니다 .
  11. 저장을 클릭합니다 .
  12. 클릭 MAP 연결로 돌아 가기페이지 상단의 링크를 클릭하여 MAP 연결 페이지로 돌아갑니다.

데이터 매핑 구성

데이터 매핑은 비디오 데이터가 Salesforce 개체의 필드에 매핑되는 방식을 결정합니다. 기본적으로 통합은 다음을 사용하여 구성됩니다. 단순한일부 비디오 데이터를 매핑하는 매핑일 Salesforce의 개체.

데이터 매핑을 사용자 지정하려면 다음 단계를 따르십시오.

  1. 매핑 유형에 대해고급을 선택합니다 .
  2. 에서 Salesforce 개체드롭 다운 목록에서 비디오 데이터를 매핑 할 Salesforce 개체를 선택합니다. 일반적으로 Pardot 통합 유형을 사용할 때보기 필드를 Salesforce 사용자 정의 개체당신이 만들었습니다.
  3. 해당보기 필드를 Salesforce 필드에 매핑합니다. 다음 예제는 사용자 정의 개체에 매핑되는 비디오 필드를 보여줍니다.
  4. 저장을클릭하여 변경 사항을 저장합니다.

Salesforce와의 통합을 제거하려면지우다 .

추적 ID 전달

Brightcove Campaign은 Salesforce에 추적 ID 값을 전달하는 기능을 지원합니다. 예를 들어 추적 ID는 캠페인 ID를 추적하는 데 사용될 수 있습니다. 추적 ID는 동영상을 표시하는 페이지 URL에 추가하거나 소스 코드에 추가 할 수 있습니다.

  • 예: URL에 추가됨
  http://www.myurl.com/video.html?
  • 예: 플레이어 소스 코드에 추가됨
  <video data-video-id="6033402539001" data-account="1486906377" data-player="default" data-embed="default" data-application-id class="video-js"  data-bc-trackingid="12345"통제 수단>< /동영상>< 스크립트 src="//players.brightcove.net/1486906377/default_default/index.min.js">< /스크립트>

그러면 Brightcove Campaign은 동기화되는 다른 필드로 Salesforce에 값을 전달합니다.

시청자 식별

뷰어를 식별하는 프로세스는 연결시 선택한 통합 유형에 따라 다릅니다.

수동 통합

Salesforce 연결이설명서통합 유형, 시청자를 식별하는 것은 귀하에게 달려 있습니다. 시청자 추적이 발생하려면 사용자 ID가 동영상을 표시하는 페이지에서 URL 매개 변수로 전달되어야합니다. 두 개의 URL 매개 변수가 지원됩니다.

  • bcemail -시청자 이메일 주소 전달
  • bcsfid -뷰어의 Salesforce ID 전달

이 시나리오에서 비디오 랜딩 페이지는 추적 목적으로 고유하게 생성 된 URL로 생성됩니다. 그러면 Brightcove Campaign은 URL 매개 변수를 사용하여 사용자 서버 측을 조회하고보기 데이터를 동기화합니다.

Pardot 통합

Salesforce 연결이 Pardot통합 유형의 경우 통합은 뷰어에 대해 Pardot 쿠키를 사용하려고 시도합니다. 이메일 주소를 사용하여 사용자를 식별하는 것도 가능합니다. 이렇게하려면 bcemail시청자의 이메일 주소가 포함 된 URL 매개 변수는 동영상을 표시하는 페이지에 전달되어야합니다.

Salesforce에서 비디오 데이터보기

비디오를 볼 때 비디오보기 데이터가 Salesforce에 동기화됩니다. 단순데이터 매핑 옵션을 선택한 경우 비디오 보기 데이터가 Salesforce 내에서작업항목으로 나타납니다. 고급데이터 매핑 옵션을 사용한 경우 매핑을 구성할 때 선택한 Salesforce 개체의 일부로 데이터 보기가 나타납니다. 다음 예에서는고급데이터 매핑이 사용되고 비디오 필드가 사용자 지정 BC Video View Salesforce 개체에 매핑되었습니다.

동영상 이름을 클릭하면 자세한 정보가 표시됩니다.
