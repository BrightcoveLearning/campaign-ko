Munchkin을 사용하여 Marketo에 연결

{{ page.description }}

브라이트코브 캠페인은 Marketo 와 통합하도록 구성할 수 있습니다. 통합이 구성되면 브라이트코브 Campaign을 사용하여 게시된 비디오의 비디오 참여 데이터가 Marketo 와 동기화됩니다. Marketo 에 들어가면 이 데이터는 세분화, 캠페인 로직, 보고서, 리드 스코어링 및 커뮤니케이션을 개인화하는 데 사용할 수 있습니다.

참고: REST API를 사용하여 Marketo와 함께 사용하도록 Brightcove Campaign을 구성하는 방법에 대한 정보는 다음을 참조하십시오. REST API를 사용하여 Marketo에 연결 .

Marketo 통합의 주요 기능은 다음과 같습니다.

  • 데이터 형식 -비디오보기 데이터는 페이지보기 또는 가상 웹 방문으로 Marketo에 전달됩니다. 웹 페이지 방문 이벤트에는 시청자의 참여 수준을 나타내는 쿼리 매개 변수가 포함됩니다. Marketo에 복제되는 데이터는 다음과 같습니다.
    • 비디오 이름
    • 비디오 ID
    • 브라이트코브 계정 ID
    • 참조 페이지의 페이지 URL
    • 브라이트코브 플레이어 ID
    • 시청률 (%)
  • 데이터 전송 -조회 데이터는 Marketo로 즉시 전송되지만 Marketo 플랫폼의 일부 보고서는 새로 고치는 데 다소 시간이 걸릴 수 있습니다.
Brightcove Campaign을 Marketo Munchkin과 함께 사용하려면 Munchkin API가 활성화 된 Marketo 계정이 있어야합니다. Munchkin 계정 ID가 필요합니다. Marketo Munchkin에 액세스이하). Munchkin 사용자 역할을 가진 사용자는 이러한 값에 액세스 할 수 있어야합니다.

Marketo Munchkin에 액세스

Brightcove Campaign과의 통합을 구성 할 때 Marketo munchkin을 입력하라는 메시지가 표시됩니다. 이 값에 액세스하려면 아래 단계를 따르십시오.

  1. Marketo 계정에 로그인하십시오.
  2. 딸깍 하는 소리관리자상단 탐색에서.
  3. 왼쪽 탐색에서관리 > 통합 > Munchkin으로 이동합니다 .
  4. Munchkin 계정 ID를복사하여 안전한 장소에 저장합니다. 통합을 구성 할 때 필요합니다.

연결 설정

다음 단계에 따라 Brightcove Campaign과 Marketo 간의 연결을 설정하십시오.

브라이트코브 캠페인과 마르케토 간의 연결 확립

  1. 브라이트코브 캠페인에로그인합니다.
  2. 탐색 헤더에서 톱니바퀴 아이콘 () 를 누른 다음 MAP 연결을선택합니다. 현재 연결 목록 (있는 경우) 이 나타납니다.
  3. 새 연결추가를 클릭합니다 .
  4. 드롭다운 목록에서 Marketo를 선택합니다.
  5. 선택대용량 (Munchkin)통합 유형.
  6. 마르케토 계정 ID를 입력합니다 . Marketo에서 복사 한 Munchkin 계정 ID입니다.
  7. 저장을 클릭합니다 .

이제 Marketo 연결이 설정되고 사용할 준비가 되었습니다. 기본적으로연결 상태는활성화됩니다 . 연결 상태는 비디오 참여 데이터가 Marketo 로 전송되는지여부를 결정합니다.

  • 활성화 - 데이터가 수집되어 Marketo 로 전송됩니다.
  • 사용 안 함 - 데이터가 수집되지 않음 (통합이 비활성화됨)

저장을클릭하여 변경 사항을 저장합니다. 통합을 제거하려면 [ 삭제] 를 클릭합니다 .
