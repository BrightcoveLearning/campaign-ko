Marketing Automation Platform 연결 구성

{{ page.description }}

브라이트코브 캠페인™ HubSpot, Eloqua 및 Marketo와 같은 인기 있는 마케팅 자동화 플랫폼(MAP)과 통합하도록 구성할 수 있습니다. MAP 통합이 구성되면 Brightcove Campaign을 사용하여 게시 된 비디오에 대한 비디오 참여 데이터가 선택한 마케팅 자동화 플랫폼에 동기화됩니다.

참고: 관리 메뉴에 액세스하려면 계정 관리자여야 합니다.
참고: 연결 프로세스에서 브라우저에서 팝업 창이 열립니다. 연결을 설정하기 전에 브라우저 팝업 차단기를 비활성화해야 합니다.

마케팅 자동화 플랫폼과의 통합을 구성하려면 다음 단계를 따르십시오.

  1. 브라이트코브 캠페인에로그인합니다.
  2. 탐색 헤더에서 톱니바퀴 아이콘 () 를 누른 다음 MAP 연결을선택합니다. 현재 연결 목록 (있는 경우) 이 나타납니다.
  3. 새 연결추가를 클릭합니다 .
  4. 연결 유형을 선택하십시오. 해당 플랫폼 연결에 대한 정보는 해당 주제를 참조하십시오.
    • Eloqua
    • Marketo ( REST API또는 Munchkin )
    • HubSpot ( REST API또는클라이언트 측 API )
    • 세일즈포스
