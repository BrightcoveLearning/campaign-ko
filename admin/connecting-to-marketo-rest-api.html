REST API를 사용하여 Marketo에 연결

{{ page.description }}

브라이트코브 캠페인은 Marketo 와 통합하도록 구성할 수 있습니다. 통합이 구성되면 브라이트코브 Campaign을 사용하여 게시된 비디오의 비디오 참여 데이터가 Marketo 와 동기화됩니다. Marketo 에 들어가면 이 데이터는 세분화, 캠페인 로직, 보고서, 리드 스코어링 및 커뮤니케이션을 개인화하는 데 사용할 수 있습니다.

참고: Munchkin ID를 사용하여 Marketo와 함께 사용하도록 Brightcove Campaign을 구성하는 방법에 대한 정보는 다음을 참조하십시오. Munchkin을 사용하여 Marketo에 연결 .

요구 사항

Marketo REST 통합과 함께 Brightcove Campaign을 사용하려면 다음을 수행해야합니다.

  • 모든 API 권한에 대한 액세스 권한이있는 Marketo에서 역할을 생성합니다.
  • LaunchPoint 서비스 생성

API 전용 사용자 및 LaunchPoint 서비스 만들기

Marketo REST API를 호출하기 전에 모든 API에 대한 액세스 권한이있는 새 API 역할을 생성하고 API 전용 사용자를 생성 한 다음 클라이언트 애플리케이션을 고유하게 식별하는 서비스를 생성해야합니다. 이러한 작업을 완료하는 단계는 Marketo 문서에 간략하게 설명되어있습니다.

연결 설정

Marketo 측의 구성 단계가 완료되면 다음 단계에 따라 Brightcove Campaign과 Marketo 간의 연결을 설정하십시오.

브라이트코브 캠페인과 마르케토 간의 연결 확립

  1. 브라이트코브 캠페인에로그인합니다.
  2. 탐색 헤더에서 톱니바퀴 아이콘 () 를 누른 다음 MAP 연결을선택합니다. 현재 연결 목록 (있는 경우) 이 나타납니다.
  3. 새 연결추가를 클릭합니다 .
  4. 드롭다운 목록에서 Marketo를 선택합니다.
  5. 표준 (REST API)통합 유형을 선택합니다. Marketo 계정에서 여러 값을 붙여 넣어야합니다. 다음 단계를 완료하려면 Marketo 관리자로 로그인해야합니다.
  6. 마르케토 계정 ID를 입력합니다 . 이 값을 얻으려면 Marketo에서관리자 > Munchkin .
  7. 당신의 Marketo REST API 끝점 URL . 이 값을 얻으려면 Marketo에서관리자 > 웹 서비스 . 복사끝점에 REST API부분.
  8. 당신의 Marketo REST API ID URL . 이 값을 얻으려면 Marketo에서관리자 > 웹 서비스 .
  9. 시작하다 Marketo LaunchPoint 클라이언트 ID . 이 값을 얻으려면 Marketo에서관리자 > LaunchPoint그런 다음세부 정보보기이 설정의 일부로 이전에 생성 한 LaunchPoint에 대한 링크.
  10. 시작하다 Marketo LaunchPoint 클라이언트 암호 . 이 값을 얻으려면 Marketo에서관리자 > LaunchPoint그런 다음세부 정보보기이 설정의 일부로 이전에 생성 한 LaunchPoint에 대한 링크.
  11. 저장을 클릭합니다 .

이제 Marketo 연결이 설정되고 사용할 준비가 되었습니다. 기본적으로연결 상태는활성화됩니다 . 연결 상태는 비디오 참여 데이터가 Marketo 로 전송되는지여부를 결정합니다.

  • 활성화 - 데이터가 수집되어 Marketo 로 전송됩니다.
  • 사용 안 함 - 데이터가 수집되지 않음 (통합이 비활성화됨)

저장을클릭하여 변경 사항을 저장합니다. 통합을 제거하려면 [ 삭제] 를 클릭합니다 .
